### ✅ Key Features

| Module | Features |
|--------|----------|
| 🔐 **Authentication** | Registration, Login, JWT Tokens, OTP via Gmail SMTP |
| 🛒 **Shopping Cart** | Add / Update / Remove items with real-time calculations |
| 💳 **Payments** | Razorpay gateway with HMAC-SHA256 signature verification |
| 📦 **Order Management** | Order tracking, status updates, delivery tracking |
| 🛡️ **Admin Panel** | Product CRUD, user management, order analytics |
| 📧 **Email Service** | OTP, order confirmation via Gmail SMTP |
